# Hotel Search App

A small frontend application for searching hotels and displaying available tour prices.
The project demonstrates working with asynchronous API requests, retries, loading and error states, and adaptive UI layout.

The focus is on clean architecture, typed data flow, and predictable state management.

---

## Tech Stack

- **React**
- **TypeScript**
- **Vite**
- **Zustand** — state management
- **CSS Modules** — styling
- **clsx** — conditional class names

---

## Features

- Search hotels by selected criteria
- Fetch prices with retry logic
- Global loading, error, and empty states
- Adaptive card grid layout
- Typed API services and hooks
